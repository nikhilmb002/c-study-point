what is a character array ?

it is a collection of characters

what is a string ?

it is a collection of characters ended with '/0'

what is '\0'

it is one of the escape sequence

'\0' indicates end of the valid data

'\0' acts as seperator between valid & invalid data

to store '\0' one byte of memory required

	* in that one byte all bits are zeros 

'\0' is not a printable character

%s is a format specifier need address while scanning as well as printing.

scanf("%s", address);

for scanning spaces : scanf("%[^\n]", address);

string length = no. of valid character
