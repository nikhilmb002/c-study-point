Function 
--------

Function is a block where program code is written.

Function is a set of instructions placed together to perform a specific task.


Benefits of function
--------------------

1. Divide and Conqueror

	* Managable program development

	* Construct a program from small pieces or components

2. Software reusability

	* Use existing function as building blocks for new programs

	* Abstraction, hide internal details


Types of function 
-----------------

1. Predefined function	/ Library function 

2. User defined function 




syntax for declaring the function 

	return_type function_name(arguments);



Depends on returntype and arguments functions can be implemented in 4 different types 

	type 1 : 
		
		void fname(void);
	
	example : 

		void abc(void);

		// abc is a function which takes nothing and returns nothing and performs one task

	type 2 : 

		void fname(arguments);

	example : 

		void def(int, char);

		// def is a function which takes 2 arguments (integer, char) returns nothing

	type 3 :

		return_type fname(void);

	example : 

		int mno(void);

		// mno is a funtion which takes nothing and returns integer

	type 4 : 

		return_type fname(int, int);

	example : 

		int sum(int, int);

		// sum is a function which takes 2 integers as an argument and return an integer






while calling the function whatever the information we are passing called as original arguments or actual arguments. In the function defnition with what we are catching that is called as formal arguments.


Actual argument name and formal argument may  be or may not be the same.
